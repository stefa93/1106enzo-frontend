# 1106enzo Design System

Welcome to the 1106enzo Design System documentation. This design system provides a comprehensive set of components, patterns, and guidelines for building consistent user interfaces.

## Overview

Our design system is built with:
- Next.js App Router
- Tailwind CSS for styling
- TypeScript for type safety
- Mobile-first responsive design

## Getting Started

To use components from this design system:

```tsx
// Import components from the web app
import { Button } from '@1106enzo/web/components';

// Use with Tailwind classes
<Button className="bg-brand hover:bg-brand-600">
  Click me
</Button>
```

## Core Principles

1. **Mobile-first Design**: All components are designed with mobile-first approach
2. **Accessibility**: WCAG 2.1 AA compliant components
3. **Performance**: Optimized for performance and bundle size
4. **Consistency**: Unified design language across all components

## Tech Stack

- **Framework**: Next.js 14 with App Router
- **Styling**: Tailwind CSS
- **Type Safety**: TypeScript
- **Testing**: Cypress, Vitest
- **Documentation**: Storybook
