## Usage Guidelines

### Component States

Use brand colors for different component states:

```tsx
// Primary actions
<button className="bg-brand hover:bg-brand-600">
  Primary Button
</button>

// Secondary actions
<button className="bg-brand-100 text-brand-900">
  Secondary Button
</button>

// Disabled states
<button className="bg-brand-100 text-brand-300" disabled>
  Disabled Button
</button>
```

### Accessibility

- Ensure sufficient contrast ratios (WCAG 2.1 AA compliance)
- Use darker shades (700-900) for text on light backgrounds
- Use lighter shades (50-100) for text on dark backgrounds

### Dark Mode

All color tokens support dark mode variants:

```tsx
<div className="bg-brand-50 dark:bg-brand-900">
  <p className="text-brand-900 dark:text-brand-50">
    Content
  </p>
</div>
```
